Questions fréquentes

Le launcher ne se lance pas, que faire ?

Vérifiez que vous avez Java 8 ou supérieur installé. Vous pouvez télécharger Java depuis le site officiel d'Oracle ou utiliser OpenJDK.

Puis-je jouer sans installer le launcher ?

Non, le launcher est obligatoire car il contient tous les mods nécessaires et les configurations spécifiques au serveur Valerium.

Le téléchargement est lent, que faire ?

Les fichiers sont hébergés sur GitHub. Si le téléchargement est lent, essayez de télécharger à un moment différent ou utilisez un gestionnaire de téléchargement.

Comment mettre à jour le launcher ?

Le launcher se met à jour automatiquement au lancement. Si une mise à jour est disponible, elle sera téléchargée et installée automatiquement.
